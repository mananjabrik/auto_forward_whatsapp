```shell
 {
 "friendNumber": [
    "friendNumber@c.us",
    "friendNumber@c.us",
    "friendNumber@c.us",
    "friendNumber@c.us"
  ]
}
```

need nodemon

```shell
npm install -g nodemon
```


```shell
npm install
```
